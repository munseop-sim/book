# git commit 규칙
| Type    | Description                           |
|---------|---------------------------------------|
| feat    | 새로운 기능에 대한 커밋               |
| fix     | 버그수정                              |
| build   | 빌드관련 파일 수정/모듈 설치 또는 삭제 |
| chore   | 자잘한 수정에 대한 커밋               |
| ci      | ci 관련설정 수정에 대한 커밋         |
| docs    | 문서 수정에 대한 커밋                |
| style   | 코드 스타일 변경에 따른 커밋         |
| refactor| 코드 리팩토링에 대한 커밋            |
| test    | 테스트 코드 수정에 대한 커밋         |
| perf    | 성능 개선에 대한 커밋                |
